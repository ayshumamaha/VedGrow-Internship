# AI-Powered FAQ Chatbot

## Overview

This project is a simple AI FAQ Chatbot developed using Python and Streamlit.

The chatbot answers frequently asked questions related to Python programming.

---

## Features

- Interactive chatbot
- Beginner-friendly responses
- Simple web interface
- Python FAQ support
- Lightweight implementation

---

## Technologies

- Python
- Streamlit

---

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```

- app.py
- chatbot.py
