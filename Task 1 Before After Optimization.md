# Before vs After Prompt Optimization

## Example 1

### Before

Write a blog on AI.

### After

Write a 1000-word SEO-optimized blog titled "The Future of Artificial Intelligence in Healthcare."

Include:

- Meta Title
- Meta Description
- H1
- H2
- H3
- Primary Keyword
- Secondary Keywords
- Conclusion

### Improvement

Provides structure, audience, formatting, and SEO requirements.

## Example 2

### Before

Write Python code for a calculator.

### After

Develop a menu-driven calculator supporting addition, subtraction, multiplication, division, modulus, and exponentiation.

Include:

- Functions
- Exception handling
- Input validation
- Comments

### Improvement

Produces cleaner and production-ready code.
