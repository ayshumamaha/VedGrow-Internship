# Document Question Answering System

## Overview

This project demonstrates a simple Document Question Answering system developed using Python.

Users can ask questions based on a predefined document, and the system retrieves the relevant information.

---

## Features

- Document-based question answering
- Simple keyword matching
- Interactive command-line interface
- Beginner-friendly implementation
- Runs without external libraries

---

## Technologies Used

- Python

---

## How to Run

```bash
python document_qa.py
```
