# AI Image Description & Alt-Text Generator

## Overview

This project demonstrates a simple AI Image Description and Alt-Text Generator developed using Python.

Users enter an image name, and the system generates a descriptive caption along with accessibility-friendly alt text.

---

## Features

- Image description generation
- Alt-text generation
- Interactive command-line interface
- Beginner-friendly implementation
- Runs without external libraries

---

## Technologies Used

- Python

---

## How to Run

```bash
python image_description.py
```
